spf_codemirror

CodeMirror syntax-highlighting and Emmet code-completion plugin for Textpattern admin.

(Emmet is the new name for Zen Coding.)

Preferences can be set for theme, font-size and full-screen hot-keys.

View themes here - to set a theme just type the name (e.g. lesser-dark) into the CodeMirror theme field and click Save.



Features

  1. Adds CodeMirror syntax-highlighting to textareas in Textpattern’s Forms, Pages and Style tabs;
  2. Also to JavaScript tab (spf_js required) and External Files tab (spf_ext required);
  3. Basic preferences via Admin > Preferences > Advanced;
  4. HTML/CSS shorthand/code-completion added to Pages, Forms & CSS courtesy of Zen Coding/Emmet.
  5. Full-screen support - just hit F5 (and Esc to exit) - or set your own hot-keys via Preferences.


Installation

  1. DOWNLOAD and unzip;
  2. Upload the containing ‘codemirror’ directory to your web root:
    • codemirror
    • css.php
    • files
    • images
    • index.php
    • rpc
    • sites
    • textpattern
  3. Install and activate the plugin (spf_codemirror.txt - inside the unzipped folder).
  4. NOTE: the folder structure mirrors that of a standard CodeMirror download (just the 'lib', 'mode' and 'theme' directories) with the addition of the minified emmet.min.js file from Emmet-CodeMirror2 - for easy upgrade.


CodeMirror/Emmet upgrade

  1. You can now easily upgrade CodeMirror (2.35 as of Nov 2012).
  2. Download the latest release.
  3. Unzip the download and upload the containing 'lib', 'mode' and 'theme' directories to the codemirror directory on your web server.
  4. To upgrade Emmet download 'Emmet-CodeMirror2', unzip it and upload the containing 'emmet.min.js' file to your codemirror directory.


Minify support

  1. If you have Minify on your web server in the standard DOCUMENT_ROOT/min location, spf_codemirror will use it to minify css & js - if you don't, it won't.


Emmet* notes

  1. Initiate completion by hitting TAB (or Cmd+E).
  2. Try typing this: div#page>div.logo+ul#navigation>li*5>a and then TAB.
  3. Works with opening and closing txp tags (try typing txp:if_section and then TAB).
  4. CSS shortcuts work even better with Emmet - try typing m and then TAB in Styles.
  5. Emmet documentation.

    6. *Emmet is the new name for Zen Coding.
